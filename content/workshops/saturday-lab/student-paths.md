# Six Student Paths — Saturday Lab

Pick **one path** at intake. Success = **one artifact + one next step** by 1 PM.

---

## Path 1 — No website (Beginner)

**Hour 1:** Site map (Google Doc). Platform choice.  
**Hour 2:** Homepage prompt → draft intro.  
**By 1 PM:** Sitemap + one page draft.  
**No account OK:** Doc only. **Account:** open Squarespace/Wix trial.

---

## Path 2 — Old website (Beginner)

**Hour 1:** Audit prompt. List clear vs confusing.  
**Hour 2:** Fix **one** page.  
**By 1 PM:** One improved page or fix list.

---

## Path 3 — Images, no writing (Beginner)

**Hour 1:** Pick 5 works + one project.  
**Hour 2:** Project description prompt.  
**By 1 PM:** Project page draft.

---

## Path 4 — Writing, no images (Beginner)

**Hour 1:** Bio prompt (50 / 100 / 250 words).  
**Hour 2:** Image shot list for 5 works.  
**By 1 PM:** Bio + shot list.  
**Continue:** [How to photograph artwork for portfolio](https://www.youtube.com/results?search_query=how+to+photograph+artwork+for+portfolio)

---

## Path 5 — Vibe Level 1 (CodePen)

**Hour 1:** Hello Browser on [CodePen](https://codepen.io/pen/).  
**Hour 2:** Compare or Debug prompt — one edit.  
**By 1 PM:** Link or screenshot.  
**Account:** optional.

---

## Path 6 — Vibe Level 2 (Replit)

**Hour 1:** New Replit HTML project OR paste starter prompt.  
**Hour 2:** Replace name, intro, one project. Mobile check.  
**By 1 PM:** Shareable Replit URL.

---

## Path 7 — Vibe Level 3 (Cursor)

**Hour 1:** [Download starter zip](/workshops/saturday-lab-starter.zip) → open in Cursor.  
**Hour 2:** Edit content + one small interaction.  
**By 1 PM:** Working local or saved project.

---

## Quick decision tree

```text
Website structure / writing / images?  → Beginner (Paths 1–4)
Try code fast, no setup?                → Vibe L1 (Path 5)
Need live link?                         → Vibe L2 (Path 6)
Comfortable with files?                 → Vibe L3 (Path 7)
Unsure?                                 → Beginner sitemap (Path 1)
```
