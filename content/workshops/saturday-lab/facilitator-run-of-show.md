# Saturday Lab — Facilitator Run of Show

> **10:00 a.m.–1:00 p.m.** · ~6 students · One instructor · Triage, not lecture  
> **Promise:** One clear next step + one working artifact per student

**Students use:** printed cheat sheets + QR → `/workshop/saturday-lab`  
**You use:** this page + whiteboard help queue

---

## Before class

- [ ] Run `npm run saturday-lab:assets` (starter zip + QR PNG)
- [ ] Print **per student:** [Start Here half-sheet](/workshop/saturday-lab/print/start-here) + track cheat sheet ([Beginner](/workshop/saturday-lab/print/beginner) or [Vibe](/workshop/saturday-lab/print/vibe-coding))
- [ ] Whiteboard help queue (table below)
- [ ] CodePen, Replit, starter zip link ready
- [ ] Sticky notes: “By 1 PM I want ______”

---

## 10:00–10:10 — Scan QR + choose output

Everyone opens the hub. They pick **one** output (write on sticky):

**Beginner:** sitemap · homepage draft · bio · project page · platform decision  
**Vibe L1:** CodePen sketch · **L2:** Replit URL · **L3:** Cursor starter edit · shareable link

Teaching line:

> “This is a lab for artists building digital presence at different levels. Beginners build structure. Vibe coders build prototypes.”

---

## 10:10–10:25 — Intake script (sort each person ~2 min)

Ask aloud or 1:1 while others open hub:

1. Are you here for **website structure**, **AI/code**, or **both**?
2. Do you already have a website?
3. What do you want to leave with — writing, sitemap, draft site, coded sketch, or **live link**?
4. Do you have accounts ready for Squarespace/Wix/Cargo/**Replit/GitHub/Cursor**?
5. Are you comfortable **editing code files**?

### Sorting guide

| Signals | Send to |
|---------|---------|
| No website, old/confusing site, images but no writing, needs bio/CV/contact | **Beginner** |
| Wants to try code without setup | **Vibe Level 1** — CodePen or local HTML |
| Wants live link | **Vibe Level 2** — Replit |
| Has files or starter zip | **Vibe Level 3** — Cursor |

Hand matching **printed cheat sheet**. Full packets on hub only.

---

## 10:25–10:40 — Shared frame (only lecture block)

**Title:** Your website is opportunity infrastructure.

**5 questions** (everyone memorizes):

1. Who are you?
2. What do you make?
3. Why does it matter?
4. What have you done?
5. How can someone contact you?

**Patterns:**

- Beginners: **Collect → Organize → Write → Build → Publish → Improve**
- Vibe: **Describe → Generate → Test → Edit → Debug → Publish**

**Vibe levels (30 sec each):**

- **L1 CodePen** — try it · **L2 Replit** — share it · **L3 Cursor** — real files  
- “You do not need to start with Cursor.”

## 11:00–12:15 — Lab (3 rounds)

Students work from cheat sheet + QR. You circulate.

### Round 1 · 11:00–11:20 — Direction only

> “What are you making by 1 PM?”

### Round 2 · 11:20–11:50 — One blocker each

Give **one next action**. Update whiteboard.

### Round 3 · 11:50–12:15 — Visible output

Something on screen or paper by 12:15.
