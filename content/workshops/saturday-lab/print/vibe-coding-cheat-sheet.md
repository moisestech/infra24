# Vibe Coding — Cheat Sheet

> **Loop:** Describe → Generate → Test → Edit → Debug → Publish  
> **You do not need to start with Cursor.**

## Three levels

| Level | Tool | Output | Account |
|-------|------|--------|---------|
| **1 — Try it** | CodePen / local HTML | Browser sketch | Optional |
| **2 — Share it** | Replit | Live link | Yes |
| **3 — Real files** | Cursor + [starter zip](/workshops/saturday-lab-starter.zip) | Edited project | Yes |

> ChatGPT & Claude = think · CodePen = test · Replit = share · Cursor = edit files

---

## No account vs account

| No account | Account |
|------------|---------|
| CodePen blank pen | Replit new HTML project |
| Open `index.html` locally | Cursor + downloaded zip |
| Paste AI code in browser | GitHub Pages (after class) |

## 3 prompts

**Starter site:** `Create a simple one-page artist website: hero, 3 projects, about, CV highlights, contact. Mobile-friendly HTML/CSS. Mark where I edit text.`

**Compare:** `Show two versions: one minimal, one more chaotic/experimental. Same content.`

**Debug:** `Something broke. Explain in plain English, point to the line, give corrected code. Do not rewrite everything.`
