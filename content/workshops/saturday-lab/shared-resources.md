## Starter template {#starter-template}

**[Download Saturday Lab Starter Template (zip)](/workshops/saturday-lab-starter.zip)**

1. Download the zip  
2. Unzip → open folder in **VS Code**, **Cursor**, or upload to **Replit**  
3. Edit `index.html` and `style.css` (follow HTML comments)  
4. Preview `index.html` in your browser  

Regenerate assets: `npm run saturday-lab:assets`
