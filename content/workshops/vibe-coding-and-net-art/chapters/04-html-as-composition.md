**Structure is visible** when you refuse to “fix” it into a generic landing template. Use headings, sections, and lists as visual rhythm—not only as SEO scaffolding.

### Sketch prompt

Use only semantic elements (`header`, `main`, `section`, `article`, `footer`, `nav`) to create a composition that feels architectural (columns implied by grouping, not by CSS grid yet).
